# 🚀 Landing Page One-Page - Desarrollo con IA

Este proyecto consiste en el desarrollo del frontend de una página de aterrizaje (landing page) de una sola página, creada a partir de una imagen de referencia utilizando herramientas de Inteligencia Artificial (**Claude AI**).

## 📋 Descripción del Proyecto
El objetivo principal fue transformar un diseño visual (suministrado en imagen) en un sitio web funcional y responsivo, aplicando etiquetas semánticas de HTML5 y estilos modernos de CSS3. 

La estructura del sitio incluye:
* **Header:** Navegación fluida y área de branding.
* **Body:** Secciones de Hero, Servicios y contenido principal basado en la imagen guía.
* **Footer:** Información de contacto y enlaces sociales.

## 🛠️ Tecnologías Utilizadas
* **HTML5:** Estructura semántica del sitio.
* **CSS3:** Estilos personalizados, diseño responsivo y layout (Flexbox/Grid).
* **IA (Claude 3.5/3):** Utilizada para la generación del código base y optimización de la estructura UI/UX.

## ⚙️ Instrucciones de Uso
1. Clona este repositorio o descarga los archivos.
2. Asegúrate de que los archivos `index.html` y `style.css` estén en la misma carpeta.
3. Abre el archivo `index.html` en cualquier navegador web moderno (Chrome, Firefox, Edge, etc.) para visualizar la landing page.
